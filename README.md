## ğŸš€ Quick Start

### Prerequisites

- A web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional, for development)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/haidrrrry/justfit.git
cd justfit
```

2. Open `index.html` in your web browser, or use a local server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

3. Navigate to `http://localhost:8000` in your browser

## ğŸ¨ Customization

### Update WhatsApp Number

Replace the placeholder phone number `923001234567` in all HTML files with your actual WhatsApp number in international format (country code + number, no spaces).

Files to update:
- `index.html`
- `about.html`
- `project.html`
- `contact.html`
- `membership.html`

### Update Contact Information

Edit `contact.html` to update:
- Email address
- Phone number
- Location/Address
- Business hours

### Change Colors

Edit `style.css` to customize the color scheme:
- Primary Red: `#DC143C`
- Accent Orange: `#FF6B35`
- Dark Background: `#0F0F0F` to `#1C1C1C`

### Replace Images

Replace images in the `/images/` folder with your own:
- `gym.jpg` - Hero and gym facility images
- Add class-specific images for better visual appeal
